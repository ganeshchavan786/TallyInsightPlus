Quick Answers

How do I get started?

Clone the repository, install dependencies with pip, configure your .env file, and run uvicorn. Check our documentation for detailed instructions.

Is AppStarter free?

Yes! AppStarter is completely free and open source under the MIT license. You can use it for any project.

How do I report a bug?

Please open an issue on our GitHub repository with details about the bug.

Can I contribute?

Absolutely! We welcome contributions. Check our contributing guidelines on GitHub to get started.
