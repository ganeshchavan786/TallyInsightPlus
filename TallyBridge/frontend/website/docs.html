Welcome to the AppStarter Framework documentation. This guide will help you understand how to use the UI components and build your application.

Installation

Clone the repository and install dependencies:

# Clone the repository git clone https://github.com/your-repo/application-starter-kit.git cd application-starter-kit # Create virtual environment python -m venv venv source venv/bin/activate # On Windows: venv\Scripts\activate # Install dependencies pip install -r requirements.txt # Run the application uvicorn app.main:app --reload --port 8501

Toast Notifications

Use the Toast API for notifications:

// Show different toast types Toast.success('Operation successful!'); Toast.error('Something went wrong'); Toast.warning('Please check your input'); Toast.info('New update available'); // Custom duration (ms) Toast.success('Saved!', 5000);

Chart.js Integration

Charts are powered by Chart.js. Include the library and create charts:

// Include Chart.js <script src="/frontend/js/lib/chart.umd.min.js"></script> // Create a chart const ctx = document.getElementById('myChart'); new Chart(ctx, { type: 'line', data: { labels: ['Jan', 'Feb', 'Mar', 'Apr'], datasets: [{ label: 'Sales', data: [12, 19, 15, 25], borderColor: '#2563eb', tension: 0.4 }] } });

Authentication API

Use the Auth object for authentication:

// Check if user is authenticated if (Auth.isAuthenticated()) { // User is logged in } // Get current user const user = Auth.getUser(); console.log(user.email, user.role); // Login await Auth.login(email, password); // Logout Auth.logout(); // Get active company const company = Auth.getActiveCompany();

Role-Based Access Control

Check user roles and permissions:

// Check if user has role if (Auth.hasRole('admin')) { // Show admin features } // Check specific permission if (Auth.hasPermission('users.create')) { // Allow user creation } // Require authentication (redirects if not logged in) Auth.requireAuth(); // Require specific role Auth.requireRole('admin'); // CSS classes for role-based visibility <div class="admin-only">Only visible to admins</div> <div class="super-admin-only">Only visible to super admins</div>
