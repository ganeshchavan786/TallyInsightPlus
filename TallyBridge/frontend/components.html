üìÖ Date & Time Pickers

Date Picker

DatePicker.init('inputId', {
  format: 'YYYY-MM-DD',
  minDate: '2024-01-01',
  maxDate: '2025-12-31',
  onChange: (date, value) => console.log(value)
});

Time Picker

TimePicker.init('inputId', {
  format: '24h',  // or '12h'
  step: 15,       // minutes interval
  minTime: '09:00',
  maxTime: '18:00',
  onChange: (time, display) => console.log(time)
});
